Programme calculant et affichant les courbes de Bézier obtenues à partir de points de controle donnés :
 - Par algorithme de De Casteljau

Compilation / exécution /affichage :
la création et exécution de l'exécutable est réalisée par le script python
python3 ShowBezierCurve.py [nom fichier points controles]
(exemple : ./bin/BezCurv.exe PtsControle2.csv)

pour lancer le programme en C uniquement :
[nom exécutable] [nom fichier points controles]
(exemple : ./bin/BezCurv.exe PtsControle.csv)
